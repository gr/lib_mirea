
 Jare Typograph
 ==============

 Для работы библиотеки типографа необходимо, чтобы на сервере был собран PHP версии 5.х.

 Помимо этого, вам необходимо добавить путь к директорию '/library/' в список
 путей для операций 'require', 'require_once', 'include' и 'include_once'. Это можно
 сделать с помощью функции 'set-include-path' (http://php.net/manual/function.set-include-path.php).

 В директории '/examples/' вы можете найти простые примеры по использованию
 данной библиотеки. Пожалуйста, обратите внимания, что для их работы может
 потребоваться некоторая настройка (как минимум, указание корректных путей
 к тем или иным файлам).
